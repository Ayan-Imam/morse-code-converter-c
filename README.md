A bidirectional Morse Code ↔ English and English ↔ Morse Code converter written in C using Struct and can handle spaces and word separation 
# Morse Code Converter in C

A console-based Morse Code converter written in C that supports:
- English to Morse Code
- Morse Code to English

## Features
- Menu-driven interface
- Supports A–Z alphabets
- Handles spaces between words
- Uses structures for Morse mapping

## Concepts Used
- C programming
- Structures
- Arrays and strings
- Loops and conditional statements
- Functions

## How to Run
```bash
gcc morse.c -o morse
./morse
